# GUD Scout - 2026-03-31

## Hallazgos principales

- **[paperclipai/paperclip](https://github.com/paperclipai/paperclip)** (+343, TypeScript) — INSTALAR — Orquestación open-source para empresas "zero-human"; encaja directo como motor de flujos automatizados de consultoría end-to-end.

- **[opendataloader-project/opendataloader-pdf](https://github.com/opendataloader-project/opendataloader-pdf)** (+55, Java) — INSTALAR — Parser PDF orientado a IA; útil para ingerir contratos, SOWs y reportes de clientes antes de procesarlos con agentes.

- **[ruvnet/ruflo](https://github.com/ruvnet/ruflo)** (+61, TypeScript) — INSTALAR — Plataforma de orquestación de agentes Claude; ofrece pipelines visuales que se pueden mapear a entregas de consultoría (diagnóstico → propuesta → reporte).

- **[thedotmack/claude-mem](https://github.com/thedotmack/claude-mem)** (+163, TypeScript) — VIGILAR — Plugin que captura automáticamente cada decisión importante en sesiones de Claude Code; ideal para mantener contexto continuo en proyectos de cliente multi-semana.

- **[Yeachan-Heo/oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode)** (+226, TypeScript) — VIGILAR — Orquestación multi-agente orientada a equipos; permite dividir tareas de consultoría entre agentes especializados (investigación, redacción, QA).

- **[revfactory/harness](https://github.com/revfactory/harness)** (+88, HTML) — VIGILAR — Meta-skill que diseña plantillas de agentes por dominio; se puede usar para generar harnesses específicos de consultoría (OKR, CRM, PPTX).

- **[mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)** (+239, Python) — VIGILAR — Skill que investiga cualquier tema en Reddit, HackerNews, etc.; automatiza la fase de benchmarking y research previo a propuestas.

- **[EveryInc/compound-engineering-plugin](https://github.com/EveryInc/compound-engineering-plugin)** (+53, TypeScript) — VIGILAR — Plugin de Office Compound para Claude Code; abre la puerta a generar documentos Word/Excel directamente desde agentes.

- **[NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent)** (+290, Python) — VIGILAR — Agente auto-adaptable ("crece contigo"); interesante como base para un agente de consultoría que aprende el estilo y contexto del cliente.

## Para tener en el radar

- **[gsd-build/get-shit-done](https://github.com/gsd-build/get-shit-done)** (+113, JS) — Meta-prompting con gestión de contexto; podría simplificar la capa de instrucciones del gud-framework.
- **[HKUDS/OpenSpace](https://github.com/HKUDS/OpenSpace)** (+69, Python) — Agentes más inteligentes y económicos a escala; referencia de arquitectura para reducir costos en pipelines largos de consultoría.
- **[ComposioHQ/awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills)** (+54, Python) — Catálogo curado de skills con integraciones externas; revisar si hay skills de Google Workspace o Notion ya listas.

## Tendencias detectadas

- **Explosión de skills y harnesses especializados**: Los repos más virales ya no son modelos sino capas de orquestación sobre Claude Code. El ecosistema se fragmenta en "harness por dominio".
- **Memoria y contexto persistente** se están convirtiendo en primitivas estándar (claude-mem, hermes-agent), lo que facilita proyectos de consultoría de larga duración.
- **Plugins de Office/documentos** empiezan a aparecer (compound-engineering-plugin, opendataloader-pdf): la brecha entre agente y entregable (.pptx/.pdf) se está cerrando.
