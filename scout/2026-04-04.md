## Tendencias detectadas

- **Explosión de skills para Claude Code**: repos como `awesome-claude-skills`, `antigravity-awesome-skills`, `agent-skills` y `Pika-Skills` apuntan a un ecosistema de skills especializados creciendo muy rápido; GUD debería publicar sus propios skills para ganar visibilidad.
- **MCP como capa de integración estándar**: `jgraph/drawio-mcp`, `epiral/bb-browser`, `tradesdontlie/tradingview-mcp` confirman que MCP se consolida como el protocolo para conectar agentes con herramientas externas; cualquier integración nueva de GUD debería exponerse como servidor MCP.
- **Pipelines "zero-human"**: `paperclipai/paperclip` y `gsd-build/get-shit-done` reflejan una demanda alta por flujos de trabajo completamente automatizados; el diferencial de GUD puede ser enfocar ese patrón específicamente en consultoría.
