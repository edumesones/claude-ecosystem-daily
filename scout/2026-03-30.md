# GUD Scout - 2026-03-30

## Hallazgos principales

- **[paperclipai/paperclip](https://github.com/paperclipai/paperclip)** (+939 semanal, TypeScript) — INSTALAR — Orquestación para "empresas sin humanos": el modelo de automatización total que gud-framework aspira a ser; arquitectura de referencia inmediata.

- **[opendataloader-project/opendataloader-pdf](https://github.com/opendataloader-project/opendataloader-pdf)** (+100, Java) — INSTALAR — Parser PDF → datos AI-ready; automatiza ingesta de propuestas, contratos y entregables de consultoría sin intervención manual.

- **[alirezarezvani/claude-skills](https://github.com/alirezarezvani/claude-skills)** (+205 semanal, Python) — INSTALAR — +192 skills y plugins listos para Claude Code; acelera el desarrollo de agentes en gud-framework sin partir desde cero.

- **[twentyhq/twenty](https://github.com/twentyhq/twenty)** (+86, TypeScript) — INSTALAR — CRM open-source moderno con API extensible; candidato directo para gestión de clientes y pipeline de proyectos en gud-framework.

- **[teng-lin/notebooklm-py](https://github.com/teng-lin/notebooklm-py)** (+205 semanal, Python) — VIGILAR — API Python + skill agéntica para Google NotebookLM; puente con Google Workspace para generación de briefings de consultoría.

- **[datalab-to/chandra](https://github.com/datalab-to/chandra)** (+105, Python) — VIGILAR — OCR que maneja tablas complejas, formularios y escritura a mano; útil para digitalizar documentos de clientes con estructuras no estándar.

- **[ruvnet/ruflo](https://github.com/ruvnet/ruflo)** (+97, TypeScript) — VIGILAR — Plataforma de orquestación de agentes específica para Claude; complementa o puede reemplazar la capa de coordinación de agentes de gud-framework.

- **[thedotmack/claude-mem](https://github.com/thedotmack/claude-mem)** (+67, TypeScript) — VIGILAR — Plugin Claude Code para captura automática de contexto; resuelve el problema de memoria entre sesiones de proyectos de consultoría largos.

- **[run-llama/liteparse](https://github.com/run-llama/liteparse)** (+47, Python) — VIGILAR — Parser de documentos rápido y open-source de LlamaIndex; alternativa ligera a opendataloader-pdf para pipelines de datos.

- **[onyx-dot-app/onyx](https://github.com/onyx-dot-app/onyx)** (+106, Python) — VIGILAR — Plataforma AI open-source con chat avanzado sobre documentos; base potencial para un portal de cliente con IA integrada.

## Para tener en el radar

- **[revfactory/harness](https://github.com/revfactory/harness)** — Meta-skill para diseñar templates de agentes por dominio; patrón aplicable a crear agentes especializados por tipo de consultoría.
- **[ComposioHQ/awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills)** — Lista curada de skills Claude; fuente de referencia para el catálogo de capacidades de gud-framework.
- **[mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)** — Skill de research sobre cualquier tema en Reddit/webs; útil para inteligencia de mercado automatizada para clientes.

## Tendencias detectadas

- **Ecosistema de skills Claude explota**: tres repos de curación/colecciones de skills en el top 50, señal de que el modelo de "skill marketplace" se consolida como estándar.
- **Orquestación multi-agente madura**: paperclip, ruflo y agentscope apuntan al mismo patrón—equipos de agentes coordinados, no agentes únicos; gud-framework debe adoptar este paradigma.
- **PDF/OCR como capa de ingesta crítica**: dos repos de parseo de documentos en trending simultáneamente; el procesamiento de documentos no estructurados es un cuello de botella resuelto que vale la pena integrar ya.
- **CRM open-source con IA**: twenty crece con fuerza; la capa de gestión de relaciones con clientes se está integrando nativamente con agentes IA.
