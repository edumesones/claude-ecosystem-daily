## Hallazgos principales

- **[paperclipai/paperclip](https://github.com/paperclipai/paperclip)** (+186 ayer / +3053 semana, TypeScript) — INSTALAR — Orquestación open-source para "empresas sin humanos"; arquitectura ideal para automatizar entregables de consultoría de punta a punta.

- **[googleworkspace/cli](https://github.com/googleworkspace/cli)** (+983 semana, Rust) — INSTALAR — CLI oficial para todo Google Workspace desde terminal; habilita automatización de Docs, Drive y Slides directamente en pipelines de gud-framework.

- **[msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents)** (+106, Shell) — INSTALAR — Agencia IA completa lista para usar (front-office a entrega); puede servir como plantilla de flujo para proyectos de consultoría.

- **[datalab-to/chandra](https://github.com/datalab-to/chandra)** (+89, Python) — INSTALAR — OCR especializado en tablas complejas, formularios y texto manuscrito; clave para ingestar documentos de clientes en pipelines de datos.

- **[opendataloader-project/opendataloader-pdf](https://github.com/opendataloader-project/opendataloader-pdf)** (+34, Java) — INSTALAR — Parser PDF preparado para IA; automatiza acceso a informes y contratos sin intervención manual.

- **[teng-lin/notebooklm-py](https://github.com/teng-lin/notebooklm-py)** (+465 semana, Python) — VIGILAR — API no oficial + skill agéntica para Google NotebookLM; abre la puerta a generar resúmenes y borradores de presentaciones desde código.

- **[virattt/dexter](https://github.com/virattt/dexter)** (+79, TypeScript) — VIGILAR — Agente autónomo para investigación financiera profunda; útil para due diligence y análisis de mercado en proyectos de consultoría.

- **[run-llama/liteparse](https://github.com/run-llama/liteparse)** (+35, TypeScript) — INSTALAR — Parser de documentos rápido y open-source de LlamaIndex; complementa chandra para procesamiento de PDFs/DOCX en pipelines.

- **[RKiding/Awesome-finance-skills](https://github.com/RKiding/Awesome-finance-skills)** (+53, Python) — VIGILAR — Colección de skills de agente para finanzas; skills reutilizables para KPIs, reportes y análisis en proyectos financieros.

- **[vercel-labs/agent-browser](https://github.com/vercel-labs/agent-browser)** (+45 ayer / +457 semana, Rust) — INSTALAR — CLI de automatización de navegador para agentes IA; permite extraer datos de portales de clientes sin API.

## Tendencias detectadas

- **Orquestación zero-human en auge**: paperclip lidera con 3K+ estrellas en la semana; el mercado valida automatización completa de agencias/consultoras.
- **Skills como ecosistema propio**: múltiples repositorios de colecciones (800+, 192+, finance, Gemini) indican que las skills se están convirtiendo en el "npm" de los agentes.
- **Procesamiento de documentos como capa crítica**: chandra + liteparse + opendataloader-pdf aparecen juntos — hay demanda clara de ingestión fiable de docs empresariales para IA.
- **Google Workspace se integra al stack agéntico**: el CLI oficial y la API de NotebookLM apuntan a que Drive/Docs/Slides serán ciudadanos de primera clase en pipelines de IA.
