# GUD Scout - 2026-04-02

## Hallazgos principales

- **[paperclipai/paperclip](https://github.com/paperclipai/paperclip)** (+117 ayer, +1.312 semanal, TypeScript) — INSTALAR — Orquestación open-source para "empresas sin humanos": pipeline completo de agentes autónomos adaptable a automatización de entregas de consultoría.

- **[phuryn/pm-skills](https://github.com/phuryn/pm-skills)** (+39, sin lenguaje) — INSTALAR — Marketplace con 100+ skills agenticas de PM: gestión de proyectos, OKRs y reportes; núcleo directo para automatizar el trabajo de consultor.

- **[thedotmack/claude-mem](https://github.com/thedotmack/claude-mem)** (+58, TypeScript) — INSTALAR — Plugin de Claude Code que captura contexto automáticamente entre sesiones; resuelve el problema de memoria de los agentes en proyectos largos de consultoría.

- **[rtk-ai/rtk](https://github.com/rtk-ai/rtk)** (+34 ayer, +353 semanal, Rust) — INSTALAR — Proxy CLI que reduce el consumo de tokens LLM un 60%; imprescindible para abaratar pipelines de agentes a escala.

- **[revfactory/harness](https://github.com/revfactory/harness)** (+40, HTML) — VIGILAR — Meta-skill que diseña templates de agentes específicos por dominio; patrón ideal para generar harnesses gud-framework por tipo de consultoría.

- **[aiming-lab/MetaClaw](https://github.com/aiming-lab/MetaClaw)** (+29, Python) — VIGILAR — Agente que aprende y evoluciona con el uso: base para construir asistentes de consultoría que mejoran solos con cada proyecto.

- **[coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills)** (+45, JavaScript) — VIGILAR — Skills de marketing para agentes (CRM, copy, campañas); reutilizable para automatizar la fase de captación de clientes en consultoría.

- **[gsd-build/get-shit-done](https://github.com/gsd-build/get-shit-done)** (+48, JavaScript) — VIGILAR — Meta-prompting y gestión de contexto ligero para agentes; útil como capa de orquestación simple antes de escalar a soluciones más complejas.

## Para tener en el radar

- **[Yeachan-Heo/oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode)** (+246, TypeScript) — Orquestación multi-agente teams-first para Claude Code; arquitectura de referencia para equipos de consultores + agentes en paralelo.

- **[kepano/obsidian-skills](https://github.com/kepano/obsidian-skills)** (+209 semanal) — Skills de Obsidian para agentes; gestión de base de conocimiento de la consultoría con acceso desde agentes.

- **[mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)** (+48, Python) — Skill que investiga cualquier tema en Reddit en los últimos 30 días; útil para research de mercado automatizado previo a propuestas.

## Tendencias detectadas

- **Explosión de skills marketplaces**: Composio, pm-skills, marketing-skills, antigravity y otros acumulan cientos de skills listas para agentes. La commoditización de skills es acelerada.
- **Reducción de coste de tokens como prioridad**: rtk, CLIProxyAPI y routers compiten para abaratar inferencia; crítico evaluar antes de escalar pipelines.
- **Memoria y contexto persistente**: claude-mem, MetaClaw y hermes-agent atacan el mismo gap — agentes que recuerdan el historial del cliente entre sesiones.
- **Multi-agente teams-first**: oh-my-claudecode y paperclip convergen en el patrón de equipos de agentes especializados en paralelo.
