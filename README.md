## 🔄 Cómo funciona

1. **GitHub Action** se ejecuta diariamente a las 9:00 AM UTC
2. **Scripts Python** recolectan datos de:
   - skills.sh (skills populares)
   - GitHub API (repos trending)
   - Hacker News (discusiones relevantes)
3. **Generación automática** de markdowns
4. **Commit y push** automático
